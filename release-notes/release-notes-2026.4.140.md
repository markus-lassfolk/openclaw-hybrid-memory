# Release Notes — OpenClaw Hybrid Memory 2026.4.140

**Date:** 2026-04-14  
**Previous baseline:** 2026.4.61

## Summary

**2026.4.140** is a **maintenance and release-alignment** drop: all published version fields (**npm** package, **OpenClaw plugin manifest**, **standalone installer**) now read **2026.4.140**, with matching **CHANGELOG** and **release notes**. The previous **`skills/personal-assistant/`** bundle is **not** included in this package anymore (only **`skills/hybrid-memory/`** remains). See **[CHANGELOG.md](https://github.com/markus-lassfolk/openclaw-hybrid-memory/blob/main/CHANGELOG.md)** section **2026.4.140** for details.

---

## Upgrade

```bash
npm install -g openclaw-hybrid-memory@2026.4.140
```

Restart the gateway after upgrading. If you use the standalone installer package, align its version with **2026.4.140** as well.
