# Release Notes — OpenClaw Hybrid Memory 2026.4.12

**Date:** 2026-04-01  
**Previous baseline:** 2026.4.11

## Summary

**2026.4.12** is a **version-only** release: bumps the npm package, `openclaw.plugin.json`, and the standalone installer to **2026.4.12** so installs and telemetry report a consistent build id. No behavioral changes relative to **2026.4.11**.

---

## Upgrade

```bash
npm install -g openclaw-hybrid-memory@2026.4.12
```

Restart the gateway after upgrading.
