# Release Notes — OpenClaw Hybrid Memory 2026.4.61

**Date:** 2026-04-06  
**Previous baseline:** 2026.4.60

## Summary

**2026.4.61** rolls up **retrieval and recall** improvements (two-phase FTS, recall pipeline telemetry and resilience, OAuth/Responses and embedding inheritance), **active tasks** facts projection, **goal stewardship** heartbeat verification and mechanical checks, and **operator** UX (goals status overview, workspace skill bootstrap). See **[CHANGELOG.md](https://github.com/markus-lassfolk/openclaw-hybrid-memory/blob/main/CHANGELOG.md)** section **2026.4.61** for the full list.

---

## Upgrade

```bash
npm install -g openclaw-hybrid-memory@2026.4.61
```

Restart the gateway after upgrading. If you use the standalone installer package, align its version with **2026.4.61** as well.
