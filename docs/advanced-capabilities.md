# Advanced Capabilities

Use this guide when you want more than baseline memory capture/recall.

## Who this is for

- Multi-agent setups
- Governance and traceability needs
- Research workflows and high-context projects

## Capability map

- Graph and relationship retrieval
- Scoped memory across global/user/agent/session boundaries
- Workflow and procedure extraction
- Crystallization and proposal-driven evolution
- Reflection and distillation loops

## Recommended starting mode

```bash
openclaw hybrid-mem config-mode complete
openclaw hybrid-mem verify
```
